## 标记语法 

##### 字体：

```
*斜体* _斜体_
**粗体** __粗体__
***粗斜体*** ___粗斜体___

```

###### 分隔线：一行只有三个以上*、-、_符号；

######  删除线：

```
~~删除文字~~
```

##### 下划线：

```
<u>带下划线文本</u>
```

##### 脚注格式:[^RUNOOB]

[^RUNOOB]:markDown yyds



##### markDown列表：

```
//无序列表
* 第一项 或 + 第一项 或 - 第一项
//有序列表
1. 第一项
2. 第二项
```

##### 区块：

```
> 文字   //第一层块
> > 文字  //第二层块
```

##### 代码标记：

```
1.反引号`代码`
2.```代码语言
	代码
```
```

```
[链接名称](链接地址)或<链接地址>
